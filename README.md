# 🏢 Egyptian Real Estate Market Analytics 2026

## 🛠 Tools & Technologies
Python | SQL Server | SSIS | Power BI

---

## 📌 Project Overview

Not every property is a good investment — but data can reveal where the real opportunities exist.

This project analyzes thousands of property listings from the Egyptian real estate market.  
Most of the dataset is based on **real 2026 listings**, with a smaller portion from previous years to ensure a more realistic view of current market trends.

The goal is not just to visualize prices, but to answer real investment questions:

- Where are the best investment opportunities?
- Which property types generate the highest returns?
- What does the market actually reward in 2026?

---

## 📊 Key Insights

- Median sale price reached **10M EGP**
- Median rental price reached **65K EGP**
- Average market rental yield reached **7.74%**

### ⚠ Key Finding

Some properties generate returns above **9–10%**, while others barely reach **3%**.

The difference is not luck — it is driven by:

- Property type  
- Unit size  
- Location  
- Real market demand  

---

## 🏢 Market Gap Analysis

The data revealed a clear market gap:

Small apartments (**60–90m²**) are significantly underrepresented, despite having strong rental demand.

👉 This makes them one of the most attractive investment opportunities in the current market.
